# AI seva Frontend

## React JS
This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

Node version : 16.17

### Setup

Go to the `frontend` folder -> 

1. `node --version` -- needs 16.17 and above
2. `npm install --force`
3. Verify details in `/src/config.js` file 
4. `npm start`
